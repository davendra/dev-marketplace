# Dev Marketplace

A collection of Claude Code skills and plugins.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add https://github.com/davendra/dev-marketplace
```

## Available Skills

### Go Bananas

AI image generation with Google Gemini models.

**Install:**
```bash
/plugin install go-bananas@dev-marketplace
```

**Features:**
- Generate images from text prompts
- Edit existing images
- Create consistent characters across scenes
- Product marketing image generation
- Style presets for branding

**Setup:**
```bash
export GO_BANANAS_API_KEY=sk_live_your_key_here
```

Get your API key at [gobananasai.com](https://gobananasai.com)
